# CN-Drop

**CN-Drop** is a secure, cross-platform file sharing application that works over your local network. No internet required.


## 🚀 Features

* **No Internet Needed:** Transfer files using local Wi-Fi.
* **Cross-Platform:** Works on Windows, macOS, Linux, Android, and iOS.
* **Secure:** End-to-end encryption for all transfers.
* **Zero Setup:** No accounts, servers, or configuration required.
